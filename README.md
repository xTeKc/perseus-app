# Prerequisites

### Rust:
```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### Perseus CLI:
```
cargo install perseus-cli --version 0.4.0-beta.10
```

<br>

# Usage

### Clone - CD - Build:
```
git clone https://github.com/xTeKc/pertau && cd pertau/ && cargo build
```

### Serve:
```
perseus serve -w
```
